# sahur_maze made by vite + three.js

미로 탈출 게임 - Three.js 기반 3D 게임

## 주요 기능
- 3인칭/1인칭 시점 전환
- 난이도 선택 (쉬움/보통/어려움)
- 실시간 스태미나 시스템
- 게임 중 홈/리플레이 버튼
